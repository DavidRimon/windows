This sample demonstrates how to use the CommandLinkWPF control from the Vista Bridge Controls library.

To build the sample

  • Run MSBuild from the sample root directory or open the Aero Wizard solution in Visual Studio and build the solution.

To run the sample

  • From the bin folder of the sample root directory run Wizard.exe.

Requirements

Vista Bridge Sample

Demonstrates

Windows Presentation Foundation (WPF)

Comments

When you download a sample from the SDK, place it in a location that you have read/write access. For example: Do not place samples in the 'Program Files' folder.

This sample has a project-to-project dependency on the Vista Bridge Sample. This configuration requires that the Vista Bridge Sample be placed next to the folder where the Aero Wizard sample resides.
