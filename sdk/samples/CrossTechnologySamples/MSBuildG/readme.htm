GUI for MSBuild is a sample demonstrating the Windows® Presentation Foundation presentation system and its ability to use the external libraries of the MSBuild Project system. It is both an example of these technologies and a useful utility for creating MSBuild projects.

To build the sample

  • Run MSBuild from the sample root directory or load the GUI for MSBuild solution in Visual Studio.

To run the sample

  1. Create a "GUI for MSBuild" folder in the "My Documents" folder.

  2. Copy all the files from the "Sample Files" folder into the "GUI for MSBuild" folder.

  3. From the MSBuildG\bin\release folder, run MSBuildG.exe.

Demonstrates

The sample demonstrates key Windows Presentation Foundation features such as data binding, animation, navigation windows, and interoperability with external libraries.
